## 🌍 The Problem
Local government officials (like Members of Parliament) receive thousands of unstructured requests daily through scattered channels (paper letters, WhatsApp, verbal complaints). It is impossible to manually process this data, leading to reactive decision-making rather than proactive, data-driven constituency development.

## 💡 Our Solution
**People's Priorities** is a dual-sided platform that digitizes and democratizes infrastructure planning. 
1. **For Citizens:** A frictionless, multilingual portal to submit anonymous needs (with voice support for low-literacy users) and track resolutions using a unique Tracking ID.
2. **For Officials:** A secure AI-powered dashboard that aggregates all requests, maps hotspots, surfaces recurring themes, and provides strategic, data-backed recommendations on how to allocate development budgets.

---

## ✨ Key Features

- **🌐 Multilingual & Inclusive:** Dynamic translation (English, Hindi, Tamil) and Voice Input for low-literacy accessibility.
- **🔒 Frictionless Anonymity:** No accounts or passwords required for citizens. Submissions generate a secure Tracking ID.
- **📊 AI-Powered Admin Dashboard:** Real-time metrics, interactive maps, and automated NLP categorization of complaints.
- **🧠 Strategic AI Recommendations:** Cross-references citizen demand with public datasets to generate actionable directives for the official.
- **🔄 Closed-Loop Tracking:** Citizens can track the exact status of their complaint (Submitted ➔ In Progress ➔ Resolved) using their ID.

## 🚀 How to Use It (Workflow)

### 👤 Citizen Flow
1. **Submit a Need:** Go to the homepage and click "Share Your Need". You can type or use the Voice Input feature. Select the category (e.g., Road Repair, Water Supply).
2. **Save your ID:** Upon submission, you will receive a unique **Tracking ID** (e.g., `5F9B2XA`).
3. **Track Status:** Return to the homepage, click "Track a Project", and enter your ID to see live updates and official responses.

### 🏛️ Govt Official Flow
1. **Login:** Access the secure portal via the "Login Portal" button on the homepage.
2. **Analyze Demand:** View the Live Map to see issue hotspots and check the Category Breakdown pie chart.
3. **Take Action:** Review the AI Strategic Recommendations.
4. **Update Citizens:** Click on an individual complaint in the feed, add an Official Response, and mark it as "In Progress" or "Resolved". This instantly updates the Citizen's tracking page!

## 💻 Local Installation

To run this project locally on your machine:

1. **Clone the repository**
   ```bash
   git clone https://github.com/reet-9944/People-s-Priorities.git
   cd People-s-Priorities
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **View the app**
   Open [http://localhost:3000](http://localhost:3000) in your browser.
